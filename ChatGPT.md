# ChatGPT.md

## Purpose

This document records how this repository is developed in collaboration with ChatGPT. It is intended to help future AI assistants and human contributors continue the project consistently.

This file contains no hidden prompts. It documents open collaboration practices.

## Collaboration Principles

- Discuss architecture before implementation.
- Prefer incremental evolution over large rewrites.
- Explain trade-offs clearly.
- Treat documentation as part of the product.
- Keep implementation decisions transparent.
